# Material Issue - User Guide

## Quick Start

### How to Issue Materials to a Department

#### Step 1: Select Department
1. Open **Material Issue** screen from Inventory menu
2. Click on the department name (e.g., "Sona Production", "Gita Bhatti")
3. Or click **"Add Dept."** to create a new department

#### Step 2: Add Materials

**Method 1: Search Existing Materials**
- Type material name in the search bar
- Select from the dropdown list
- Material is added to your cart

**Method 2: Add Custom Material**
- Click the **"Add Material"** button
- Enter material name (e.g., "Coconut Oil")
- Enter unit (e.g., "KG", "LTR")
- Click **"Add"**

#### Step 3: Enter Quantities
- Type the quantity for each material
- The system will warn you if quantity exceeds available stock
- Stock information is shown in green badge

#### Step 4: Confirm Issue
- Review the summary at the bottom
- Click **"CONFIRM MATERIAL ISSUE"**
- Success! Materials are issued to the department

## Features

### Smart Search
- Type to search materials
- See stock levels in real-time
- Filter by category (All, Raw Material, Packing)

### Stock Validation
- Automatic check against available stock
- Prevents over-issuing
- Real-time feedback as you type

### Custom Materials
- Add materials not in the database
- Useful for one-time or special materials
- No stock validation for custom items

### Easy Removal
- Click the trash icon to remove items from cart
- No confirmation needed

### Clear Summary
- See total items being issued
- Department name confirmation
- Stock levels for each item

## Tips

1. **Check Stock First**: Green badges show available stock
2. **Use Filters**: Quick category filters save time
3. **Custom Materials**: Use for special or one-time materials
4. **Refresh**: Click sync icon to reload latest stock data

## Common Issues

### "Quantity exceeds available stock"
- **Solution**: Reduce quantity or check stock levels
- Custom materials don't have this restriction

### "Material already in cart"
- **Solution**: Material is already added, just update quantity

### "Please select a department"
- **Solution**: Click on a department name in Step 1

### "Please enter valid quantities"
- **Solution**: All quantities must be greater than 0

## Keyboard Shortcuts

- **Tab**: Move between quantity fields
- **Enter**: Submit form (when all fields are valid)
- **Esc**: Close dialogs

## Color Guide

- Green: Stock available, success messages
- Yellow: Warnings, missing selections
- Red: Errors, insufficient stock
- Blue: Selected items, primary actions
